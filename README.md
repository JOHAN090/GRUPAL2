# Proyecto Grupal Multimedia

Este repositorio contiene los ejercicios correspondientes al proyecto grupal de multimedia.

## Ejercicio 1 (Sistema con PHP)

El **Ejercicio 1** (carpeta `ejer1mult`) está desarrollado utilizando **PHP**, por lo que requiere un servidor web con soporte para PHP (como **XAMPP**, **WAMP** o **MAMP**) para funcionar correctamente. **No se puede visualizar directamente a través de GitHub Pages.**

### Instrucciones para ejecutar el Ejercicio 1 localmente (con XAMPP):

1. Debes tener instalado [XAMPP](https://www.apachefriends.org/es/index.html).
2. Descarga o clona este repositorio en tu computadora.
3. Copia toda la carpeta del proyecto (`GRUPAL`) y pégala dentro de la carpeta `htdocs` de XAMPP. La ruta por defecto en Windows es: `C:\xampp\htdocs\`.
4. Abre el panel de control de XAMPP y presiona "Start" en el servicio **Apache**.
5. Abre tu navegador web y accede a la siguiente dirección: 
   `http://localhost/GRUPAL/ejer1mult/` 
   *(Si cambiaste el nombre de la carpeta, ajusta la URL en consecuencia).*

## Ejercicios 2 y 3

Los demás ejercicios están desarrollados con tecnologías web estáticas (HTML, CSS, JS, WebGL, etc.) y pueden visualizarse sin problemas directamente desde GitHub Pages o simplemente abriendo el archivo `index.html` en cualquier navegador web.
